팀이름 : Late But Lucky
프로젝트 이름 :　Mate Launcher
담당 교수님 : 이민석
팀원 : 김권섭(팀장),김성현,나동규,류종원

프로젝트 내용:
Mate Launcher는 Android Launcher로써
홈화면을 자신의 방을 꾸미듯 다양한 이미지 아이콘으로 꾸밀 수 있습니다.
아이콘을 특정 앱과 매칭하여 어플리케이션을 실행하거나 폴더로 사용,
아바타는 특정 연락처와 매칭하여 홈화면에서 특정 연락처에게 재미있고 빠르게 SMS,SNS를 보내고 받을 수 있습니다. 
런처 홈페이지에 꾸민 홈화면을 자랑하고, 다른사용자의 홈화면을 다운받아 내 스마트폰에
적용시킬 수 있는 등 다른 사용자와 커뮤니케이션을 할 수 있습니다.


타켓머신 :　안드로이드 스마트폰(Gingerbread version)
타켓운영체제 : Android
개발도구 : Eclipse, Adobe Photoshop, git, google App engine
개발언어 :　Android, Java
